### Daily Hydration
- Minimum 80oz water
- 1000mg Vitamin C (500mg morning, 500mg evening)
- 15g collagen morning (optional 10g evening)

### Emergency Decompression Reset
For rib slips - DO NOT push the rib back:
1. Lie on right side
2. Reach left arm overhead
3. Deep breath into left flank
4. Let gravity + breath reset position

### Baby Lifting Strategy
- Use step stool by crib
- Step up before lifting
- Engage lats first
- Never twist while holding
